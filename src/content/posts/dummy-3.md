# 함수형 프로그래밍 입문

불변성과 순수 함수, 고차 함수의 개념을 실제 코드로 연습했다. map, filter, reduce를 활용한 데이터 처리에 재미를 느꼈다.

## 실습 코드

```typescript
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((n) => n * 2);
```
